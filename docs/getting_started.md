# What is LogXPro?

LogXPro is an autonomous, open-source Security Operations Center (SOC) designed to automatically ingest logs, correlate events into attack chains using Sigma rules, and generate actionable incident baskets.

## Core Features
1. **Automated Ingestion:** Directly parses Windows Event Logs, Syslog, Zeek, and PCAP.
2. **Correlation Engine:** Groups isolated events into unified "Kill Chain" baskets.
3. **AI Narrative Generation:** Automatically writes a human-readable summary of the attack.
4. **GRC Profiles:** Maps detections and playbooks to specific industry regulations (e.g., Healthcare, Finance).
